An example of this style is on this page: 
http://www.scripts.oldguy.us/talkback/demo/style4.php

style4test.php is a example/test page. You do not need to put it in your TalkBack
directory unless you want to use it for testing.

To use this style:
	Copy to the TalkBack directory:
		my-comments-display-tpl.php
		my-comments-form-tpl.php
		my-style.css
	Change the paths of URL's in the css if they not correct for your installation:
		replace all "/talkback/images"  with  "/path-to-talkback/images"
	Change the file name of the templates in the configuration settings
	Author website link:
		To use the template as is, change "Show author website" to no link
		If you want to show the author's website link:
			Change configuration setting "Show author website"  to  name link
