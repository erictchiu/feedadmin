Comment sort order
Comments are listed on comment pages either ascending or descending (visitor choice) by date and time except that reply comments are always listed under the original comment in ascending sequence.

Comments in the admin panels are sorted by date and time whether or not they are replies.

Comment replies
If you want to make an existing comment be a reply to another comment, edit it and enter the ID number of the original comment in the reply to field. The ID number of comments is shown in the admin panel listings.

To make a reply become an original comment, edit it and clear the reply to field.

Bypass admin menus
To eliminate going through the admin panel menus to do a backup, create a bookmark with a URL of http://yoursite.com/talkback/admin/?action=backupdb. You can also optimize the database this way with ?action=optimizedb.
