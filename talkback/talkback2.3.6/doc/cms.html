Creating a guestbook in a Content Management System

Table of Contents

You can use TalkBack as a guestbook on a website that is based on a Content Management System (CMS). You will, however, have to learn how to connect a static page (the guestbook page) to your system’s menu.

This is the procedure for Mambo and Joomla.

Create a guestbook.php and upload it to the directory of your choice. See the example in addons/example-mambo-guestbook.php.

In the Mambo administration panels create a new main menu item of type “wrapper”. The URL will be http://yoursite.com/directory/guestbook.php. Set the height field to at least 1000 if not more.
